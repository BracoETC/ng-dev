{{ title }} String Interpolation uses double curly braces to bind values from the VM (View Model) to the view.
Expressions can be used to do dynamic evaluation, calculation, formatting like:
Calculation: 1 + 3 + {{ nbr }} = {{ 1 + 3 + nbr }}
Ternary operator: Value of 'isFalse': {{ isFalse ? 'flasch' : 'nicht falsch' }}
Function Call: {{addNumbers(10, myObj.a)}} Read more on Template Syntax on Angular Docs
